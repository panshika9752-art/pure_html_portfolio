# Enquiry Form (HTML Project)

This is my first project on GitHub 

## Project Description
This project is a simple Enquiry Form created using HTML.  
It collects basic user information like:
- Name
- Address
- Gender
- Country
- Hobbies
- Email
- Password
- Date of Birth
- Age
- Image upload

## Technologies Used
- HTML5

## Purpose
I created this project to practice HTML form elements and understand how user input works.

## Features
- Input fields (text, email, password, number)
- Radio buttons for gender selection
- Checkbox for hobbies
- Dropdown for country selection
- File upload option
- Submit and Reset buttons

(Simple form UI in browser)
